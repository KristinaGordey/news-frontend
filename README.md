# Frontend (Vue 3 + Vite)

Frontend-часть приложения, реализованная на **Vue 3** и **Vite**,  
с управлением состоянием через **Pinia** и аутентификацией через **Strapi v5 (JWT)**.

---

## Стек технологий

- Vue 3 (Composition API, `<script setup>`)
- Vite
- Pinia
- Axios
- Vue Router

---

## Установка зависимостей

npm install

## Запуск проекта

npm run dev
